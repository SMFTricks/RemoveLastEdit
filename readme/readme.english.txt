[size=4][color=#bf307a]Introduction[/color][/size]
[size=1em]Remove Last Edit allows users to easily remove the "Last edit by" signs, with the press of a button.[/size]

[size=3][color=#1f4882]Features[/color][/size]
[list]
[li][size=0.9em]Doesn't remove the whole Last Edit section, it just adds a button.[/size][/li]
[li][size=0.9em]Access to the button is set with a permission.[/size][/li]
[li][size=0.9em]Redirects you back to the post in question after removing the sign.[/size][/li]
[li][size=0.9em]Doesn't alter the post itself.[/size][/li]
[/list]
